HTML

HTML stands for HyperText Mark-up Language. It is used on the frontend and gives the structure to the webpage which you can style using CSS and make interactive using javascript.

1. The DOCTYPE html tag declaration defines that this document is an HTML5 document

2.The html tag element is the root element of an HTML page

3. The head tag element contained meta information about the HTML page

4. The title tag element specific a title for the HTML page(which is shown in the browser's tite bar or in the page's tab)

5.The body tag elements defines the documenrt's body, and is a container for all the visible contents, such as headings, paragraph, images, hyperlinks, tables, lists, etc.

6. The h1 tag elements defines a larger heading

7. The p tag element defines a paragraph


CSS

CSS orCascading Style Sheets is the language used to style the frontend of any website. CSS is a cornerstone technology of the World Wide We, alongside HTML and JavaScript.

1. CSS stands for Cascading Style Sheets

2. CSS describes how HTML elements are to be displayed on screen, paper, or in other media

3. CSS saves a lot of work. it can control the layout of multiple web pages all at once

4. External stylesheets are stored in CSS files.
